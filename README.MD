# Secure Task and Document Management System

## Overview
This project is a secure, role-based task and document management system designed for enterprise environments. It features encrypted storage, comprehensive audit logging, and robust authentication mechanisms to ensure data integrity and security.

## Key Features
- **Role-Based Access Control (RBAC):** Granular permissions to manage user access effectively.
- **Encrypted Storage:** Secure handling of documents and sensitive data at rest.
- **Audit Logging:** Detailed logs of system activities for compliance and monitoring.
- **Enterprise-Style Authentication:** Secure identity verification and management.

## Technology Stack
- **Backend:** C# ASP.NET Core
- **Frontend:** Blazor
- **Language:** C#
